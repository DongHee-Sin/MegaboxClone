# 라이징캠프 5주차 과제
#### 클론코딩한 앱 : 매가박스
#### 기간: 2022/03/05 ~ 2022/03/12

<br/>

## 소셜로그인과 Open AP를 활용한 클론코딩 프로그래밍
### 조건
* Alamofire사용하여 API 최소 3개 이상 연동
* 소셜로그인 최소 1개 이상 구현 (+ 로그인한 유저 정보를 활용)

<br/>

### 사용한 API
* 영화진흥위원회 : 일별 박스오피스 조회 API
* 네이버 : 영확 검색 API
* 카카오 소셜로그인 API

<br/>

### 구현된 화면
* 홈 View
* 스토어 View
* 예매 View
* 모바일오더 View
